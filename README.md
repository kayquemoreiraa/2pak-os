# 🚀 2Pak OS

Sistema operacional interno da 2Pak Studio para gerenciamento de clientes, prospecção, tarefas e projetos.

## 📋 Funcionalidades

- 📊 Dashboard
- 🏢 Gestão de Empresas
- 👤 Gestão de Contatos
- 📅 Gestão de Tarefas
- 📝 Observações
- 🎯 Pipeline Kanban
- 🔍 Pesquisa Global
- ⏰ Follow-up de Leads

## 🛠️ Tecnologias

### Backend
- Node.js
- Express
- MySQL
- mysql2
- UUID

### Frontend
- React
- Vite
- Tailwind CSS

## 📂 Estrutura

```
backend/
frontend/
docs/
```

## 🎯 Objetivo

Centralizar toda a operação da 2Pak Studio em uma única plataforma, desde a prospecção de clientes até a gestão dos projetos.
